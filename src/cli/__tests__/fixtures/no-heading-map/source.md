# Game Theory

This lecture introduces basic game theory concepts.

## Nash Equilibrium

A Nash equilibrium is a state where each player has chosen an optimal strategy
given the strategies of the other players.

## Prisoner's Dilemma

The prisoner's dilemma is one of the most classic examples in game theory.
